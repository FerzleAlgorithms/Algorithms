Polynomial Evaluation

Problem Description

Given the coefficients of a polynomial p(x) = anxn + … + a1x + a0, and a value x, compute p(x).

Input

An array of coefficients [a0, a1, …, an] (from constant term up to the highest-degree term), and a number x.

Output

The value of the polynomial p(x).

Motivation/Applications

Polynomial evaluation is central in numerical analysis (e.g., curve fitting, approximation), computer graphics (e.g., computing Bézier curves), and scientific computing. Efficient evaluation—especially for high-degree polynomials—can greatly reduce computation time.

Examples

Example 1
Evaluate Polynomial p(x) = 2x3 – 6x2 + 2x – 1 at 3.
Input: Coefficients array: [-1, 2, -6, 2]
x = 3
Output: 5
Explanation: 2·3³ – 6·3² + 2·3 – 1 = 54 – 54 + 6 – 1 = 5.

Example 2
Evaluate polynomial p(x) = x2 – 4 at 2.
Input: Coefficients array: [-4, 0, 1]
x = 2
Output: 0
Explanation: 1·2² + 0·2 – 4 = 4 – 4 = 0.

Common Algorithms/Techniques

  • Naïve evaluation (compute each xi separately, O(n²) time)
  • Better evaluation (compute each xi by multiplying previous power by x, O(n) time)
  • Horner’s method (rewrite in nested form, O(n) time, slightly faster than the "better" one)
  • Divide-and-conquer splitting (parallel evaluation, useful for SIMD/GPUs)

Variants/Related Problems

  • Polynomial interpolation (e.g., Newton’s or Lagrange’s methods)
  • Simultaneous evaluation at multiple points (use FFT-based techniques)
  • Derivative evaluation (compute p′(x) alongside p(x))
  • Modular polynomial evaluation (compute p(x) mod m)
  • Evaluation of multivariate polynomials
