Chain Matrix Multiplication

Problem Description

Given a sequence of matrices A₁, A₂, ..., Aₙ, where each matrix Ai has dimensions pi−1 × pi, determine the most efficient way to multiply the matrices together. The goal is to fully parenthesize the product A₁A₂...Aₙ to minimize the total number of scalar multiplications.

Input

A sequence of integers p₀, p₁, ..., pₙ such that matrix Ai has dimensions pi−1 × pi. The number of matrices is n.

Output

The minimum number of scalar multiplications required to compute the matrix product, along with an optimal parenthesization (optional).

Motivation/Applications

Matrix chain multiplication arises in many applications involving linear algebra and optimization, such as graphics transformations, dynamic programming, and scientific computing. While matrix multiplication is associative, the order can greatly affect computational efficiency.
