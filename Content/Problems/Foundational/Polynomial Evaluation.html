Polynomial Evaluation

Problem Description

Given the coefficients of a polynomial \(p(x) = a_n x^n + \dots + a_1 x + a_0\), and a value \(x\), compute \(p(x)\).

Input

An array of real number coefficients \(\bigl[a_0, a_1, \dots, a_n\bigr]\) (from constant term up to the highest-degree term), and a number \(x\).

Output

The value of the polynomial \(p(x)\).

Motivation/Applications

Polynomial evaluation is central in numerical analysis (e.g., curve fitting, approximation), computer graphics (e.g., computing Bézier curves), and scientific computing. Efficient evaluation—especially for high-degree polynomials—can greatly reduce computation time.

Examples

Example 1
Evaluate the polynomial \(p(x) = 2x^3 - 6x^2 + 2x - 1\) at \(x = 3\).
Input:
Coefficients array: \([-1, 2, -6, 2]\)
\(x = 3\)
Output: 5
Explanation: \(2\cdot3^3 - 6\cdot3^2 + 2\cdot3 - 1 = 54 - 54 + 6 - 1 = 5\).

Example 2
Evaluate the polynomial \(p(x) = x^2 - 4\) at \(x = 2\).
Input:
Coefficients array: \([-4, 0, 1]\)
\(x = 2\)
Output: 0
Explanation: \(1\cdot2^2 + 0\cdot2 - 4 = 4 - 4 = 0\).

Common Algorithms/Techniques

  • Naïve/brute-force evaluation (compute each \(x^i\) separately, \(O(n^2)\) time)
  • Better evaluation (compute each \(x^i\) by multiplying the previous power by \(x\), \(O(n)\) time)
  • Horner’s method (rewrite in nested form, \(O(n)\) time, uses about \(n\) multiplications instead of the \(2n\) multiplications used by the previous one.)

Variants/Related Problems

  • Polynomial interpolation (e.g., Newton’s or Lagrange’s methods)
  • Simultaneous evaluation at multiple points (use FFT-based techniques)
  • Derivative evaluation (compute \(p'(x)\) alongside \(p(x)\))
  • Modular polynomial evaluation (compute \(p(x)\bmod m\))
  • Evaluation of multivariate polynomials
