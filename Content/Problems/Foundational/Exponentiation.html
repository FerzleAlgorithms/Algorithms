Exponentiation

Problem Description

Given a base \(a\) and a non-negative exponent \(n\), compute the exponentiation \(a^n\).

Input

Two integers: a base \(a\) and an exponent \(n\), where \(n \geq 0\).

Output

The value of \(a^n\), i.e., \(a\) multiplied by itself \(n\) times.

Motivation/Applications

Exponentiation is a fundamental operation in mathematics and computer science. It underlies growth models, complexity analysis, and cryptographic protocols (e.g., modular exponentiation in RSA), and efficient power computation via exponentiation by squaring is key in many algorithms.

Examples

Example 1
Input: a = 2, n = 10
Output: 1024
Explanation: 210 = 1024.

Example 2
Input: a = 5, n = 3
Output: 125
Explanation: 53 = 125.

Common Algorithms/Techniques

  • Naïve loop multiplication (\(O(n)\) time)
  • Naïve Divide-and-conquer (\(O(n)\) time)
  • Binary Exponentiation (also called Exponentiation by squaring, Fast Exponentiation, and other names) (\(O(\log n)\) time)

Variants/Related Problems

  • Modular exponentiation (compute \(a^n\bmod m\))
  • Exponentiation with negative exponents (computing reciprocals)
  • Matrix exponentiation for solving linear recurrences
  • Real-number exponentiation and power functions
  • Computing roots (the inverse of exponentiation)
