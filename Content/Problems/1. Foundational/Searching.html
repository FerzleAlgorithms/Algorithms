Searching (an array)

Problem Description

The array searching problem involves determining whether a specific integer is present within a given array.

Input

An array of integers and a single integer value (the target value) to search for within the array.

Output

The index of the target integer if it exists in the array; otherwise, return -1 to indicate that the element does not exist.

Motivation / Applications

Array searching is fundamental in computer science and widely used in applications such as database querying, search engines, inventory management, and real-time data retrieval systems. Efficient searching algorithms enhance performance significantly in systems handling large volumes of data.

Examples

Example 1
Input: Array = [3, 8, 2, 7, 5], Target = 7
Output: 3


Example 2
Input: Array = [1, 4, 9, 6], Target = 2
Output: -1

Common Algorithms / Techniques

  • Sequential Search (also called Linear Search)
  • Binary Search (if the array is sorted)

Variants / Related Problems

  • Searching in sorted arrays
  • Searching in linked lists
